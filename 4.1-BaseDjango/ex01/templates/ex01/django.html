Ex01: Django, framework web.

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. It was originally developed in 2003 by the web programming team at the Lawrence Journal-World newspaper and released publicly in 2005. Django provides an ORM, templating, routing, authentication and many other batteries-included components.
