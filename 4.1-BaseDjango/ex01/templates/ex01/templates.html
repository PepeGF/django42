Ex01: Template engine.

The Django template engine merges templates with context data to produce HTML. It supports template inheritance with blocks, control structures such as for loops and if conditions, and variable display from the context passed by the view.

Blocks and inheritance

Templates can extend a base template and override named blocks (e.g. title, style, content).
