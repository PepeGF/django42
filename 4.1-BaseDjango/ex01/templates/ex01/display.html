Ex01: Display process of a static page.

This page describes the flow when Django renders a simple template:

  1. The browser sends a request to a URL (e.g. /ex01/display/).
  2. The project's URLconf routes the request to the app's urls.py, which selects the appropriate view function.
  3. The view prepares any context data and calls render(request, template, context).
  4. Django's template engine loads the template file, combines it with the context, and produces the final HTML.
  5. The view returns an HttpResponse with that HTML back to the browser.
